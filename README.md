# Svelte Icon Webkit
A collection of reusable Svelte components for building user interfaces in Svelte SVG Icon Sets web applications.

- CodeWrapper
- DynamicCodeBlockStyle
- Footer
- helpers
- HighlightCompo
- HomeCards
- IconPage
- MetaTag
- Nav
- SupportBanner
- TechInfo


## Requirements

- svelte 5 (Runes)

## Installation

```sh
pnpm i -D svelte-icon-webkit svelte-rune-highlight highlight.js
```
